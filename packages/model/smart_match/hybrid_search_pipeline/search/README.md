### search 폴더

- **`fusion_retriever.py`**
  - 이미지 검색 결과와 텍스트 검색 결과를 가중합으로 결합하는 `HybridFusionRetriever`를 제공합니다.
  - `FusionWeights(alpha, beta)`를 통해 이미지/텍스트 점수 비율을 조정할 수 있으며, 교차 인코더 자리는 현재 `noop` 구현으로 비워두고 있습니다.
  - 검색 결과에는 모델 메타데이터, OCR 텍스트, 이미지 경로 목록, 유사도 점수 등이 포함됩니다.

#### 검색 흐름
1. 이미지 질의가 있을 경우 BGE-VL로 임베딩을 생성하고 Milvus 이미지 컬렉션에서 ANN 검색을 수행합니다.
2. 텍스트 질의가 있으면 BGE-M3로 임베딩 후 모델 컬렉션에서 검색합니다.
3. 모델별로 이미지/텍스트 점수를 병합하고, part number 필터링 및 정렬을 거쳐 최종 결과를 반환합니다.

#### 에이전트 연동
- 오픈월드 에이전트는 위 검색을 “tool”로 호출한 뒤, 웹에서 정보를 보강하고(가격 포함) 필요 시 모델 메타데이터를 갱신합니다.
