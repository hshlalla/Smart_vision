# 7. References

이 파일은 최종 보고서의 참고문헌 정리용 작업 파일이다.

## 사용 원칙

- 최종 reference list는 기존 draft와 제출용 문서의 citation을 기준으로 정리한다.
- citation style은 최종 제출 형식에 맞춰 일관되게 통일한다.
- 본문에 실제로 인용한 자료만 남긴다.
- 웹 자료는 가능한 한 문서명, 기관명, 접근일을 포함해 정리한다.

## 정리 템플릿

아래 형식은 예시다. 실제 제출 기준 스타일에 맞게 수정하면 된다.

1. Author, A. A. and Author, B. B. (Year) *Title*. Publisher or Journal, volume(issue), pages.
2. Organisation Name (Year) *Page or document title*. Available at: URL (Accessed: DD Month YYYY).
3. Author, A. A. (Year) *Conference paper title*. In: Conference Name, pages.

## 작업 메모

- eBay image search, Google Lens, OCR robustness, vector databases, human-in-the-loop AI 관련 reference를 본문 citation과 맞춘다.
- 최종 조립 시에는 numbering, punctuation, italic 처리, URL 스타일을 한 번에 정리한다.
- draft에 있던 reference 번호와 새로 추가한 문헌이 충돌하지 않게 다시 번호를 매긴다.
